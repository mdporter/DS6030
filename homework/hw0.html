c. Install the course quarto extension

This document will not compile until you load the ds6030hw-html quarto extension. To do this, open a terminal window (if using RStudio, look for the terminal tab) and navigate to the same directory where this file is. Then run:

quarto add mdporter/quarto-ds6030-hw

which should create an _extensions directory. This contains the formatting for the compiled html document.
